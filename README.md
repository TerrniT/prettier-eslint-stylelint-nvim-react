# Getting Started

## Prettier

`npm i --save-dev -save-exact`

`nvim .prettierignore`

`nvim .prettierrc.json`



## Eslint

`npm i -D eslint`

`npm i -D eslint-config-prettier`

`nvim .eslintignore`

`nvim .eslintrc.js`



## Stylelint

`npm i -D stylelint stylelint-config-standard`

`npm i -D stylelint-config-prettier`

`npm i -D stylelint-lsp`
